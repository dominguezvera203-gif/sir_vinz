For the API-Quest project, I engineered a multisource intelligence dashboard that orchestrates five distinct data streams—CoinGecko, NewsData.io (local and global), NASA’s APOD, and Open-Meteo—into a unified "Intel Feed." The app solves the problem of information fragmentation by consolidating disparate signals—from Bitcoin market pulses and Mati City weather to international tech news—into a single, tactical interface, eliminating the need for users to jump between multiple platforms for a daily briefing. The most difficult part of the integration was managing the asynchronous synchronization of these concurrent requests; using Promise.all required careful data normalization to map five different JSON structures into a consistent internal schema while ensuring a single API failure wouldn't crash the entire system. With more time, I would transition to Promise.allSettled to improve service resilience, allowing the dashboard to partially load even if one provider is offline, and implement a caching layer to respect rate limits while providing a faster, near-instantaneous user experience.
